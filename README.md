# Expense Tracker

A clean and minimal expense tracker web app built using vanilla HTML, CSS, and JavaScript.

## Features

- Add and delete expenses
- Categorize expenses
- Filter by category
- Sort by newest, oldest, highest, or lowest amount
- Automatic total expense calculation
- Category spending breakdown
- LocalStorage support (data persists in browser)
- Responsive modern UI
- Expand/collapse long descriptions
- Toast notifications for actions

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## How It Works

The app stores all expenses in the browser using `localStorage`.  
Users can:

1. Enter amount, category, and description
2. Add expenses instantly
3. View total spending and category breakdown
4. Filter and sort expenses dynamically
5. Delete single expenses or clear all data

No backend or database is required.

## Run Locally

1. Download or clone the repository

```bash
git clone https://github.com/your-username/spent-expense-tracker.git
```

2. Open the project folder

3. Run the app by opening:

```bash
expense_tracker.html
```

in any modern browser.
